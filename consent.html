Política de Privacidade e Tratamento de Dados

Este sistema coleta informações pessoais exclusivamente para fins de registro, controle de tentativas, auditoria e análise de desempenho do quiz corporativo. Nenhum dado é compartilhado com terceiros.

Os dados coletados incluem: nome, e-mail corporativo, telefone, respostas do quiz e registros de consentimento. Os dados são armazenados com base na LGPD - Lei Geral de Proteção de Dados (Lei 13.709/2018).

Ao aceitar, você concorda com:
• Armazenamento seguro das suas informações
• Uso dos dados exclusivamente para fins internos
• Registro das tentativas de avaliação
